# Relay

Relay 是一个基于 Codex CLI 的 Web 工作台，用来管理 AI 对话、会话、记忆与自动化执行。

## 核心价值

- 让长期任务在多轮对话中保持连续推进。
- 把关键会话沉淀成可复用的记忆资产。
- 通过自动化让目标在当前回复之后继续执行。

## 产品功能概览

### 首页 / 关于

![Relay 首页](docs/images/home-about.png)

首页清晰展示了 Relay 的核心工作流：`Session -> Memory -> Automation`。

### 工作区

![Relay 工作区](docs/images/workspace.png)

工作区是主操作面板，支持继续对话、查看上下文与文件联动处理。

### 会话

![Relay 会话](docs/images/sessions.png)

会话页面用于组织历史任务，帮助用户识别哪些内容值得长期保留。

### 记忆

![Relay 记忆](docs/images/memories.png)

记忆页面将关键对话整理为结构化信息，方便后续检索与复用。

### 自动化

![Relay 自动化](docs/images/automation.png)

自动化页面支持围绕目标配置长期任务，让 AI 从“一次回答”走向“持续执行”。

### 设置

![Relay 设置](docs/images/settings.png)

设置采用配置文件驱动，便于以 TOML 方式统一管理产品行为与界面参数。
